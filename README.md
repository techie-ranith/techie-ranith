### Howdy!

I'm a web designer and developer with a passion for crafting elegant, user-centric applications. As a full-stack developer specializing in Python Flask and modern web technologies, I focus on building scalable CRUD applications and RESTful APIs. I'm the co-founder of several open-source projects and actively contribute to the developer community.

I've been coding for several years, giving workshops on back-end development and database design. Along the way, I've built multiple production-ready applications, collaborated on Flask projects, and continuously expanded my expertise in cloud technologies. When I'm not coding, you'll find me exploring new frameworks, contributing to open source, or sharing knowledge through technical writing.
